Python: https://python.org
PyGame: https://www.pygame.org 
Install on other version: https://github.com/renpy/pygame_sdl2

OpenCV-Python (Replacement for pygame.movie): https://pypi.org/project/opencv-python

(Optional)
NumPy (for array in PyGame): https://pypi.org/project/numpy
PyOpenGL (for animation, 3D model in PyGame): https://pypi.org/project/pyopengl
MoviePy, PyGame Video (for movie playing only in PyGame): https://pypi.org/project/moviepy, https://pypi.org/project/pygamevideo
PyGame SDL2 (for replacement with PyGame): https://test.pypi.org/project/pygame_sdl2
PyGame SDL2 build: (
    Code: https://github.com/renpy/pygame_sdl2,
    SDL2: https://github.com/renpy/pygame_sdl2_windeps
    )

Py2Exe (require in code version): https://pypi.org/project/py2exe

(Linux/Unix)
Executing the game.py to run the game.
